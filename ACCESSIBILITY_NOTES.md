# Accessibility Notes

This site is being aligned to the practical standards referenced by:

- Illinois DoIT accessibility guidance under the IITAA
- WCAG 2.1 Level AA
- Federal ADA Title II web accessibility rule language that points public entities to WCAG 2.1 AA

Implemented in this repo:

- semantic page landmarks
- visible skip link
- labeled form controls
- keyboard-focus styles
- reduced-motion support
- mobile navigation that works without hover
- clear headings and section structure
- alt text on instrument photos
- live status messaging in the multi-step form

Still recommended before production launch:

- manual keyboard-only walkthrough of every page
- screen reader check with VoiceOver or NVDA
- automated scan with axe or Lighthouse
- color-contrast verification for any future brand changes
- caption/transcript policy for any future audio or video embeds

This is implementation guidance, not legal advice or a formal legal certification.
